## Бизнес-проблема:
Компания теряет потенциальных клиентов из-за отсутствия проактивного подхода в работе с пользователями. Маркетинговые усилия не оптимизированы, так как нет возможности предсказать, какие пользователи с высокой вероятностью совершат целевые действия.

## Бизнес-проблема:
Разработать ML решение, которое в реальном времени анализирует поведение пользователей на сайте и предсказывает вероятность совершения конверсионных действий до их фактического выполнения.
